 # 🤖 بوت XAUUSD — مساعد تداول الذهب

بوت Telegram يرسل تنبيهات تلقائية خلال جلسات London و New York Killzone مع Checklist ICT كاملة وإدارة مخاطر احترافية.

## ✅ المميزات

- تنبيه تلقائي عند دخول Killzone
- Checklist ICT كاملة: Liquidity Sweep، FVG، OB، SMT
- تحذير عند وجود أخبار USD قوية
- إيقاف تلقائي عند خسارتين متتاليتين أو 3% يومياً
- حساب المخاطرة 1% لكل صفقة

## 🚀 طريقة التشغيل

1. ثبّت المكتبات: pip install requests python-dotenv
2. انسخ .env.example وسمّه .env
3. ضع توكنك الحقيقي في .env
4. شغّل: python xauusd_bot.py

## ⚠️ تنبيه

لا تشارك ملف .env مع أي شخص
البوت مساعد فقط — القرار النهائي لك أنت
